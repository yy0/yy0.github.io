自定义JSTL函数标签

当JSTL自带的标签无法实现我们的需求时，就需要写自定义标签来实现我们需要的功能。

以替换字符串中指定字符为例。例如将字符中以“-*”结尾的字符替换成“以上”。
1、首先编写JAVA工具类

public class StringFunction
{
    /**
    *
    * @param source
    * @return
    */
    public static String replace(String source)
    {
        if (source.endsWith("-*"))
        {
            source = source.replace("-*", "以上");
        }
        return source;
    }
}

2、编写tld文件
str.tld
<?xml version="1.0" encoding="UTF-8" ?>

<taglib xmlns="http://java.sun.com/xml/ns/j2ee" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:schemaLocation="http://java.sun.com/xml/ns/j2ee http://java.sun.com/xml/ns/j2ee/web-jsptaglibrary_2_0.xsd" version="2.0">

    <description>km functions library</description>
    <display-name>km functions</display-name>
    <tlib-version>1.0</tlib-version>
    <short-name>sf</short-name>
    <!-- 自己定义的URL,页面中引用 -->
    <uri>http://www.test.com/functions</uri>
    <function>
        <!-- 页面中调用函数名称 -->
        <name>replace</name>
        <function-class>com.test.StringFunction</function-class>
        <function-signature>java.lang.String replace(java.lang.String)</function-signature>
    </function>
</taglib>

3、页面使用
页头引入自定义标签
<%@ taglib uri="http://www.test.com/functions" prefix="sf" %>
标签中使用：
<span>${sf:replace(testString) }</span>
